Div (division)

The DIV (division) element is a block element, and will always start on a new line and take up the full width available (stretches out to the left and right as far as it can). Most common and generic container element.

<div style="border: 1px solid black;text-align: center;><p>"Hello World</p></div>

Hello World

HTML with external stylesheet

Links the CSS file using the filepath relative to the html document referencing it.

    <head>
      <linkrel="stylesheet" href="styles.css">
    </head>

Attributes

Class

Identifier the can be used on one or more elements. Can be used by stylesheets and javascript.

      <h4>My <span class="note">Important</span> Heading</h4>
      <p>This is some <span class="note>important</span> text.<p>

My Important Heading

This is some important text.

You can also attach multiple classes to a single element

      <p>This is with <span class="note>one</span> class.<p>
      <p>This is with <span class="note large highlight>multiple</span> classes.<p>

This is with oneclass.

This is with multiple classes.

Id

Same as class but must only be used on one HTML element

<h4 id="myHeader>My header with id attribute</h4>

Placeholder

The placeholder attribute specifies a short hint that describes the expected value of an input field (e.g. a sample value or a short description of the expected format).\ The short hint is displayed in the input field before the user enters a value. 

      Name: <input type="name" id="name" name="name" placeholder="First and Last name">
      Email: <type="email" id="email" name="email" placeholder="example@domain.com">
